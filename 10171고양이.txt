public class Main {
	public static void main(String[] args){
        System.out.println("\\    /\\");
        System.out.println(" )  ( ')");
        System.out.println("(  /  )");
        System.out.println(" \\(__)|");
    }
}



- " (큰따옴표)와 \(역슬래쉬)는 그냥 출력되지 않아서 앞에 \(역슬래쉬)를 한번더 붙여줘야 원하는대로
  출력이 된다.

- 그리고 꼬리 모양이 예제문제와 같지 않아서 (띄어쓰기 때문에) 출력 형식이 잘못되었다고 떴다.
